CMTR2 RNA-seq Analysis Pipeline

This repository contains the complete analysis pipeline for RNA-seq data comparing Kras/Trp53/CMTR2 vs Kras/Trp53 conditions in Mus musculus (GRCm38/mm10).

Overview: <br />
14 samples (A1-A14) <br />
A1-A7: Kras/Trp53 (KP) <br />
A8-A14: Kras/Trp53/CMTR2 (KPC) <br />
Mouse reference genome: GRCm38/mm10 <br />

Pipeline Steps
1. FastQC Quality Control (1. fastqc.sh) <br />
Raw RNA-seq reads in FastQ files were first subjected to quality control using FastQC version 0.12.1 (Andrews, 2023) to assess read quality, base composition, GC content, and sequencing artifacts. <br />

2. STAR Alignment (2. staralignment.sh) <br />
Reads that were determined to be high quality by FastQC were aligned to a STAR genome index using the Spliced Transcripts Alignment to a Reference (STAR) software version 2.7.10a (Dobin, 2013) to produce coordinate-sorted BAM files. The STAR genome index was generated using the GRCm39 (mm39) mouse reference genome and the GRCm39 GENCODE gene annotations (with Ensembl, Havana, and Ensembl-Havana merged transcripts) with a splice junction overhang parameter of 149 to match the read length used in sequencing. Alignment quality and transcriptomic coverage were evaluated using Qualimap version 2.2.1 (Okonechnikov, 2016). <br />
   
3. Qualimap (3. qualimap.sh) <br />
RNA-seq specific quality metrics were calculated using Qualimap’s “rnaseq” module, which assesses features including per-transcript coverage, junction sequence distribution, and genomic localization of reads. Analyses were performed using the same mouse gene annotation file that was used during alignment. The libraries were treated as non-strand- specific paired-end data. <br />
   
4. Feature Counting (4. featurecount.sh) <br />
Gene-level read counts were quantified using featureCounts from the Subread version 2.0.2 package (Liao, 2021). Reads were assigned to genomic features using the mouse gene annotation file. Only reads that overlapped annotated exons were counted and summarized at the gene level. The resulting gene count matrix was generated from the coordinate-sorted STAR BAM files. <br />
  
5. Differential Expression Analysis (5. Differential Expression Analysis.R) <br />
Differential expression analysis was performed on the featureCounts matrix in R using the DESeq2 package (Love, 2014), which uses a generalized linear model with a negative binomial distribution to model count data, estimate log2 fold changes, and identify differentially expressed genes between conditions. DESeq2 also normalizes counts by the median-of-ratios method, which divides the raw counts by sample-specific size factors determined by the median ratio of counts to geometric mean for each gene (Anders, 2010). The median-of-ratios normalization method takes into consideration both sequencing depth and RNA composition to allow for inter-sample count comparisons. Log2 fold changes were estimated for KPC samples relative to KP samples as the reference condition. Positive log2 fold change values indicate higher gene expression in KPC tumors, whereas negative values indicate lower gene expression in KPC tumors. Gene identifiers were mapped from Ensembl IDs to gene symbols using the org.Mm.eg.db annotation database (Carlson, 2026). <br />

   To evaluate sample relationships and global transcriptomic variation, principal component analysis (PCA) was performed. A variance-stabilizing transformation (VST) was first applied to the DESeq2-normalized count data to reduce mean-variance dependence, such that variance does not depend on mean expression level. PCA was performed using the top 1000 most variable genes across all samples. Sample clustering was visualized using ggplot2 (Wickham, 2016), and cluster separation between conditions was quantitatively assessed via silhouette scores based on Euclidean distance in PCA space using the cluster R package (Maechler, 2026). <br />

Heatmaps of gene expression were created using the pheatmap R package (Kolde, 2025). Expression values from the variance-stabilized dataset were extracted for the top 1000 genes ranked by adjusted p-value, scaled by row (z-score transformation), and visualized across samples. The row ordering of the heatmap was based on hierarchical clustering of the scaled gene expression values, using Euclidean distance to group genes with similar expression patterns across samples. Silhouette analysis was applied to the heatmap gene expression matrix to quantify clustering strength between KP and KPC samples. <br />

An MA plot was created using ggplot2 (Wickham, 2016) to visualize the relationship between DESeq2-normalized mean gene expression and log2 fold change. Genes were classified as significantly and substantially differentially expressed if they satisfied the thresholds of adjusted p-value < 0.05 (Degen, 2025), absolute log2 fold change ≥ 1 (Degen, 2025), and normalized mean expression across all samples ≥ 10 (Love, 2025). The mean expression threshold was set to filter out genes with relatively low expression that could lead to noise and false positives (Lin, 2016). Genes that satisfied the differential expression thresholds were highlighted on the MA plot and labeled using gene symbols. The fold change threshold and mean expression threshold were also marked on the plot. <br />

To further visualize differential expression results, a bar plot was generated to show the log2 fold changes for genes that satisfied the differential expression thresholds. A heatmap of only the differentially expressed genes was constructed using variance-stabilized expression values to illustrate expression patterns separating KP and KPC samples and to validate differential expression. The row ordering of the heatmap was based on hierarchical clustering of the z-scaled gene expression values, using Euclidean distance to group genes with similar expression patterns across samples. Principal component analysis was performed on only the set of differentially expressed genes to evaluate how strongly these genes separated the KP and KPC experimental groups. Silhouette score based on Euclidean distance was calculated for both the heatmap and PCA plots to quantify clustering strength between KP and KPC samples. <br />

To investigate functional pathways associated with transcriptional changes, Gene Set Enrichment Analysis (GSEA) was performed using the clusterProfiler R package. Genes were ranked by their DESeq2-calculated Wald statistic, which divides log2 fold change by standard error, incorporating both effect size and variance. Ensembl gene IDs were stripped of version suffixes to standardize IDs, and duplicate gene entries arising from this step were collapsed by retaining the entry with the largest absolute Wald statistic. After mapping Ensembl IDs to Entrez IDs using the org.Mm.eg.db annotation database (Carlson, 2026), any duplicate gene entries were similarly resolved by retaining the entry with the largest absolute Wald statistic. <br />

Using the Entrez IDs, GSEA was conducted using three primary curated pathway databases: Gene Ontology (GO) (Ashburner, 2000; The Gene Ontology Consortium, 2025; Thomas, 2022), Kyoto Encyclopedia of Genes and Genomes (KEGG) (Kanehisa, 2019, 2000, 2025), and Reactome (Fabregat, 2017). These three databases are each widely used in bioinformatics research and are often used in combination to provide complementary coverage of gene functions, pathways, and molecular interactions (Good, 2021; Muley, 2025; Nguyen, 2024; Zhou, 2025). GO annotates genes in the categories of Molecular Function (activities performed by genes), Cellular Component (location where a function takes place), and Biological Process (programs accomplished by multiple molecular activities together). Only the Biological Process (BP) ontology was used, as it captures pathway-level activities and cellular processes more directly linked to cancer development than Molecular Function or Cellular Component annotations. KEGG consists of pathways related to metabolism, genetic information processing, environmental information processing, cellular processes, organismal systems, human diseases, and drug development. Reactome focuses on biological reactions between entities, including nucleic acids, proteins, and therapeutics, that facilitates the grouping of entities into pathways. Gene sets containing fewer than 10 genes were excluded to reduce noise. Enrichment results were visualized using dot plots displaying the most significantly enriched gene sets. <br />
